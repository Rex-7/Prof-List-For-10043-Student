## 为什么这个项目需要你？

导师的职位、研究方向、主页甚至招生情况（名额）每年都在变化。一个静态的列表很快就会过时。

本项目的价值**不**在于收集成千上万的教授名字，而在于**维护一个高质量、高时效性的“精选”列表**。

## 🎯 如何贡献？(我们的重点)

我们**最需要**的贡献是：

1.  **修正错误信息：** 比如教授的职称已变动、主页链接已失效、研究方向不准确等。
2.  **添加关键备注：** 这是最有价值的信息！例如 `今年招PhD`, `对10043友好`, `组里氛围好`, `刚入职缺学生`, `今年名额已满` 等。

你不必懂 Git 或编程。你只需要在 Google Sheet 中使用“**建议模式**”来提交你的修改，维护者审核通过后即可生效。

详细的提交流程，请**务必阅读**我们的 **[贡献指南 (CONTRIBUTING.md)](CONTRIBUTING.md)**。
